## What is WebHub?

**WebHub Desktop** is a lightweight desktop workspace built with Electron that turns your most-used web services into a single, focused application.

Instead of juggling dozens of browser tabs, extensions, and windows, WebHub keeps everything in one place:

* Chat apps
* Video platforms
* AI tools
* Music / media services
* Productivity websites
* Custom bookmarks and dashboards

Built for people who live in the browser — but want a **cleaner, faster, more focused desktop experience**.

## ✨ Key Features

### ⚡ Persistent Web Tabs

Sites stay loaded in memory for instant switching.

No waiting for Discord / YouTube / Gemini / other services to fully reload every time.

---

### 🖼 Live Compact Preview Windows

Open lightweight mini versions of websites directly on the dashboard.

Perfect for:

* quick monitoring
* music control
* chat glance
* background services

---

### 🔖 Smart Bookmark System

Built-in bookmark manager with quick access.

Organize your workspace around the services you actually use.

---

### 🎛 Full UI Customization

Customize:

* themes
* backgrounds
* layout
* dashboard blocks
* appearance settings

Make it yours.

---

### 🍪 Cookie Import

Import cookies from JSON for faster login/session recovery.

Useful for advanced users and account portability.

Helpful for services that behave differently across devices.

---

### 🎵 Global Media Hotkeys

Control playback across embedded services:

* Play / Pause
* Skip
* Media focus

Without switching tabs.

---

### 🔌 Proxy Support

Optional proxy configuration for network compatibility and custom routing.

---

### ⚙ Automation Hooks

Built-in scripting / automation hooks for advanced workflows.

Power-user friendly.

\## 📖 Подробная инструкция (для пользователей)



\### 1. Как запускать сайты

\* \*\*Навигация:\*\* В нижней части окна расположены иконки сервисов (YouTube, Discord, Gemini). Нажмите \*\*ЛКМ\*\* для быстрого переключения.

\* \*\*Мини-версии:\*\* На стартовом экране можно открыть мобильную версию сайта прямо в маленьком окне, нажав на иконку \*\*ПКМ\*\*.

\* \*\*Полная версия:\*\* Нажмите \*\*ЛКМ\*\* по иконке на стартовом экране для открытия сайта во все окно.

\* \*\*Статус-бар:\*\* Для управления окнами используйте статус-бар: \*\*ЛКМ\*\* — свернуть/развернуть, \*\*ПКМ\*\* — закрыть окно и выгрузить сайт из памяти.



\### 2. Управление и горячие клавиши

\* \*\*Меню управления:\*\* Нажмите клавишу \*\*Alt\*\*, чтобы вызвать верхнюю панель (Файл, Вид, Настройки).

\* \*\*Обновить страницу:\*\* Если сайт завис, нажмите \*\*Ctrl + R\*\*.

\* \*\*Разработчикам:\*\* Открыть консоль отладки можно через \*\*Ctrl + Shift + I\*\*

\* \*\*Меню кастомизация\*\* по горячей клавише \*\*q\*\*, что открыть меню настроек стартовой страницы.



\### 3. Работа через прокси

\* Программа поддерживает работу через прокси (настройки задаются в `main.js`). Это позволяет обходить региональные ограничения для встроенных сервисов.



\### 4. Возможные проблемы с авторизацией в Google

\* \*\*Решение:\*\* В меню управления выберите агент \*\*"mobile"\*\*, откройте Google Drive в отдельном окне (нажатием на \*\*колёсико мыши\*\*) и выполните вход. После этого авторизация сохранится для остальных сервисов.



\## ✨ Основные возможности

\* \*\*Мультизадачность:\*\* Быстрый доступ к мессенджеру, видеохостингу и ИИ в одном интерфейсе.

\* \*\*Proxy Support:\*\* Встроенная поддержка прокси-серверов.

\* \*\*Кастомизация:\*\* Гибкая настройка внешнего вида и User Agent.
